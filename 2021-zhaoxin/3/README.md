## 实现方式

定义全局的互斥锁和条件变量，以及 `cur_char`，表示当前需要输出的字母。

在 `func` 中，最外层是一个 $100$ 次的循环，表示 `a ~ z` 要输出 $100$ 次。

每次循环内，定义一个 `unique_lock`，局部锁住互斥锁，随后条件变量要求等到 `cur_char` 等于 `ch` 时，再输出 `ch`，并更新 `cur_char`，条件变量 `notify_all`。
